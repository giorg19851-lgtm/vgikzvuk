# ВГИК тренер: HTML/PWA приложение

Формат: чистые HTML + CSS + JS. Сервер не нужен.

## Запуск на телефоне

Лучший вариант:
1. Залей всю папку на GitHub Pages, Netlify или Vercel.
2. Открой ссылку с телефона.
3. В Safari/Chrome выбери «Добавить на экран Домой».
4. После первого открытия приложение будет работать как PWA.

## Быстрый запуск на компьютере

Можно просто открыть `index.html`.  
Но PWA-офлайн режим и service worker работают только через `https://` или `localhost`.

Локальный сервер:

```bash
python3 -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```

## Логика тренера

- Один вопрос на экран.
- Бесконечный тест.
- Последовательность случайная, но связанная:
  - чаще вопросы той же темы;
  - учитываются общие теги;
  - иногда вставляются вопросы из других тем.
- При ошибке вопрос возвращается через 5-10 вопросов.
- Прогресс хранится в `localStorage`.
- Открытые вопросы проверяются самопроверкой: «Показать ответ», потом «Верно» или «Неверно».

## База

База встроена в `db.js` как:

```js
window.VGIK_DB = {...}
```

Так меньше шансов, что телефонный браузер устроит клоунаду с загрузкой JSON.
