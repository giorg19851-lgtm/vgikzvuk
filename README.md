# ВГИК тест v2

Это версия без режима «напиши ответ».

Теперь каждый вопрос превращается в тест с вариантами:
- если варианты уже были в базе, используются они;
- если вопрос был открытый, приложение само подбирает 3 ложных ответа из базы;
- после нажатия показывает правильный ответ и пояснение;
- ошибка возвращается через 5-10 вопросов.

Визуал упрощён:
- меньше заголовков;
- без текстового поля;
- без лишнего меню на экране;
- чёрно-белая графика, тонкие вертикальные линии, немного чернильной фактуры.

Для GitHub Pages клади содержимое этой папки в корень репозитория:
- index.html
- style.css
- app.js
- db.js
- manifest.json
- service-worker.js
- assets/
